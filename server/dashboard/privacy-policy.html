Privacy Policy

ChatGPT Usage Tracker Extension

This Privacy Policy describes how the ChatGPT Usage Tracker Chrome extension collects, uses, and stores information for internal enterprise usage reporting.

Information We Collect

  • Windows system username of the signed-in user
  • GPT name being used
  • Conversation ID and message ID
  • Turn number
  • First question summary and current question summary
  • Session ID and timestamps

How Information Is Collected

The extension runs on ChatGPT pages and captures limited usage metadata from supported GPT conversations. On managed Windows systems, a local native messaging host is used only to obtain the currently signed-in Windows username so records can be associated with the correct employee account.

How We Use the Information

  • To create internal usage reports for supported GPT tools
  • To identify which employee session generated a tracked interaction
  • To support reporting, auditing, and operational analytics
  • To troubleshoot failed or duplicate logging events

Where Data Is Sent

Usage data is transmitted over HTTPS to the organization-managed reporting server for storage and dashboard reporting.

What We Do Not Do

  • We do not sell personal information.
  • We do not use the extension for advertising purposes.
  • We do not download or execute remote code through the extension.

Data Sharing

Data is intended for internal business use only and is accessible only to authorized personnel responsible for reporting, analytics, administration, or support.

Data Retention

Data is retained according to internal business and operational requirements. Retention periods may be adjusted by the organization.

Security

Reasonable administrative and technical safeguards are used to protect stored data and data transmitted to the reporting server.

Contact

For questions about this Privacy Policy or the extension deployment, contact your internal IT or system administrator.

Effective date: April 4, 2026
